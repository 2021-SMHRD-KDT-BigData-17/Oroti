# Oroti
# 주제 : MLP와 고용노동부 빅데이터를 활용한 장애인 직업 추천 시스템 및 구인구직 서비스
   ##  개발 목표
       • MLP모델과 장애인 경제활동 실태조사 데이터를 활용하여 직업추천 서비스 구현
       • 구인자와 구직자간 상호 소통이 가능한쪽지페이지 구현
       • 장애인 결제활동 실태조사 데이터를 기반으로 장애별, 직군별, 성별 3가지 요소를 분류 후, 인구수 차트 구현
